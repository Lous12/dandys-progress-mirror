# Dandy’s Progress v12

Неофициальный фанатский трекер прогресса для Dandy’s World.

## Что добавлено в v12

- favicon для вкладки браузера и ярлыка на телефоне;
- `site.webmanifest` и иконки 32/180/192/512 px;
- административные бейджи профилей;
- собственное название бейджа, эмодзи, цвет и описание;
- бейджи показываются в карточках игроков и публичном профиле;
- скрытая панель управления доступна только владельцу, добавленному в `site_admins`;
- выдача и удаление защищены RLS-политиками Supabase;
- обновление бейджей через Realtime.

## Обязательное обновление

1. Сначала должна быть установлена миграция v11 (`supabase-social-v11.sql`).
2. Выполни `supabase-badges-v12.sql` в Supabase SQL Editor.
3. Назначь свой User UID администратором отдельной строкой из конца SQL-файла.
4. Перенеси рабочий `supabase-config.js`.
5. Загрузи все файлы сайта, включая `badges.js` и favicon-файлы.

## Безопасность

- кнопка администратора скрыта интерфейсом, но главная защита находится в RLS базы;
- обычный пользователь не может выдать, изменить или удалить бейдж через API;
- не добавляй `service_role` или secret key в файлы сайта.
