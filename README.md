# Dandy’s Progress v5

Фанатский трекер прогресса для Dandy’s World с личными аккаунтами и открытыми страницами игроков.

## Быстрый запуск без аккаунтов

Открой `index.html`. Коллекция, мастерства, исследования и достижения будут сохраняться в браузере.

## Что есть в v5

- Личная страница прогресса.
- Все туны, мастерства, твистеды, исследования, тринкеты и достижения.
- Ручные счётчики машин, метров, предметов и особых заданий.
- Более игровой фанатский дизайн в стиле Gardenview.
- Регистрация и вход через Supabase.
- Отдельное приватное облачное сохранение для каждого пользователя.
- Раздел **«Игроки»** с поиском открытых профилей.
- Публичная страница игрока с коллекцией, мастерствами, исследованиями и общей статистикой.
- Выбор туна для аватара, короткое описание и ссылка вида `#player=nickname`.
- Переключатель приватности публичной страницы.
- Локальный гостевой режим, экспорт и импорт JSON.

## Важно

Для аккаунтов и страниц игроков:

1. Заполни `supabase-config.js`.
2. Запусти актуальный файл `supabase-setup.sql` в SQL Editor проекта Supabase.
3. Опубликуй папку сайта на Cloudflare Pages.

Полная инструкция находится в `DEPLOY.md`.
