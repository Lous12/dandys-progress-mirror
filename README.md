# Dandy’s Progress v11

Неофициальный фанатский трекер прогресса для Dandy’s World.

## Что добавлено в v11

- новая палитра Gardenview: тёплая бумага, зелёная клетка, тёмно-фиолетовое меню, цветные таблички и толстые контуры;
- отдельная вкладка **«Друзья»**;
- поиск пользователей по нику и отображаемому имени;
- входящие и исходящие заявки;
- добавление, удаление, блокировка и разблокировка игроков;
- личные сообщения длиной до 500 символов;
- непрочитанные сообщения и счётчик в меню;
- сообщения обновляются через Supabase Realtime, плюс есть резервное периодическое обновление;
- русский и английский интерфейс социальных функций;
- друзья могут видеть профиль друг друга даже при выключенной публичности.

## Важно перед публикацией

1. Перенеси рабочий `supabase-config.js` из старой версии.
2. Обязательно выполни `supabase-social-v11.sql` в Supabase SQL Editor.
3. Загрузи все файлы сайта, включая новый `social.js`.
4. Для GitHub можно не заменять уже настроенный `supabase-config.js`.
5. После публикации обнови сайт через `Ctrl + F5`.

## Безопасность

- RLS ограничивает дружбу и переписки участниками.
- Писать друг другу могут только принятые друзья.
- Блокировка запрещает новые заявки и сообщения.
- Сообщения не имеют сквозного шифрования; не используйте чат для чувствительных данных.
- В открытые файлы разрешено помещать только publishable/anon key. Никогда не публикуй `service_role` или secret key.
